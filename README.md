# WGS_varroa_samples_2025

Pipeline for the analysis of whole-genome Apis mellifera sequencing data.
Reads where generated with Illumina NovaSeq 2 x 150 bp to a targeted depth of 240x.
The high targeted depth is intented because of the pool-seq design of the samples.
Every sample consists of 30 worker bee heads. The DNA was extracted for each sample in 5 Extractions of 6 Heads. The isolated DNA was then equimolarly pooled.

The pipeline consists of scripts for each of the main parts. The basic structure was taken from the Kooperative Project in Göttingen (https://github.com/papluta/Kooperativ2023/tree/master/NGS%20data%20wrangling).
